# Pewo

## Summary

- ファイルマネージャー (スマホアプリ)
- クローズドソース
- 個人のサーバーやPCのファイルをスマホで閲覧する想定
- Samba+スマホのファイラーの代替となることを目指す

## Challenges

- 普通のファイラーだとSamba経由でサムネイルの表示が遅い
  - サーバー側でサムネイルを生成して送信し、表示速度を改善したい
- 普通のファイラーだと再生できる動画に制限がある
  - サーバー側で変換後に送信し、非対応フォーマットが再生できない問題を解決したい
